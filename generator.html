Job Autofill

Use It On Mobile

Android Chrome

  1. Upload `autofill.js` and paste its public URL into the hosted script URL field.
  2. Build the bookmarklet and copy the short bookmarklet URL.
  3. Create any bookmark, edit it, set the name to Job Autofill, and paste the copied URL into the URL field.
  4. When a job form opens in browser, tap the address bar, type Job Autofill, then tap the bookmark suggestion.

iPhone Safari

  1. Upload `autofill.js` and paste its public URL into the hosted script URL field.
  2. Build the bookmarklet and copy the short bookmarklet URL.
  3. Add this page as a bookmark, then edit that bookmark and replace its address with the copied URL.
  4. Open the job form in Safari, open bookmarks, then tap Job Autofill.

LinkedIn app pages sometimes block injected scripts. If autofill does nothing, open the application page in Chrome or Safari and run the bookmarklet there.
