Job Autofill

One bookmark for filling repeated job application forms on mobile.

Setup On Mobile

What To Do

  1. Upload this folder to GitHub Pages.
  2. Open this page on your phone.
  3. Tap the copy button above.
  4. Create any browser bookmark named Job Autofill.
  5. Edit that bookmark and replace its URL with the copied text.
  6. Open a job form, type Job Autofill in the address bar, and tap the bookmark.

Important

This uses

If this folder is public on GitHub, your details inside autofill.js are public too.
